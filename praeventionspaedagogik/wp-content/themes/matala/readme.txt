== Description ==

"Matala": http://wordpress.org/extend/themes/matala, version 1.2
Converted by Automattic for WordPress.com, July 2011
Integrated back into standalone version, July 2011 - Version bumped to 1.5.7 to reflect wp.com version number

Matala is a colorful, textured theme that supports two widget areas, six post formats, and several customization options, including custom background, custom header and an optional random image gallery on single image pages.

You can also access the theme files via Subversion:

* WordPress.org version: http://themes.svn.wordpress.org/matala/1.5.7
* WordPress.com version: https://wpcom-themes.svn.automattic.com/matala/
