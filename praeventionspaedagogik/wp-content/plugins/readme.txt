This plugin allows you to change the header size requirements in the new Twenty Eleven theme without modifying any code.

== Installation ==

Extract the zip file and upload the contents to the wp-content/plugins/ directory of your WordPress installation and then activate the plugin from plugins page. 

The Resize Header function is available under the "Appearance" tab. You will gain the ability to resize any new images you upload.
